What is Bitwisr?

Bitwisr: i2CS is an introductory curriculum designed to get people started in coding by teaching them skills in HTML, CSS, JavaScript, and Python. By combining educational videos, interdisciplinary lessons, and extra resources into a free and easy to use curriculum, it allows anyone with access to the Internet to learn to teach computer science.

Bitwisr works by combining two web platforms to host and develop educational content: GitHub and YouTube. GitHub is a collection of online repositories (think folder on Dropbox or Google Drive) for source code. People can create their own repositories and host their own code to share with the world. YouTube is a website where people can upload videos (sometimes about cats).

Download & Adapt

It's easy to use Bitwisr! Download the materials from GitHub and watch the videos on YouTube.

Hey Teachers! Have an idea about how to make the lessons even better? Just add your new ideas directly into the existing content. If you think it's an idea so good everyone should see it, you can even submit a request to have your changes added into the source lessons!

Bitwisr@Home

Bitwisr lessons can be used by individuals to learn to code on their own. Lessons include built in pacing guides, resources for more information, and no special or expensive software is required.

Bitwisr@School

Bitwisr lessons can also be used in the classroom as a base curriculum for computer science education. Whether you are a teacher or a tutor, you can take advantage of the learning objectives and supplemental resources for your students.
