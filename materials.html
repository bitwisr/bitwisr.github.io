Getting started with Bitwisr

Easiest

Download the materials right here! Click the button below to download the contents of the Bitwisr repository as a ZIP file. Extract the files to a folder and you're good to go!

Download Lessons


Easy

Download the materials on GitHub.

  1. Click here to go to the Bitwisr repository on GitHub.
  2. In the sidebar, to the right of the 'Objectives' heading, click the 'Download ZIP' button.
  3. Extact the contents of the ZIP file.


Pro (for contributors)

Fork the Bitwisr repository and create your own local clone. Use this method if you want to contribute your own changes to the source repository.

  1. Create a fork of the Bitwisr repository.
  2. Clone your fork to your local machine.
  3. Use and edit the materials in your local clone.

How to contribute

Contributing to the Bitwisr repository is easy and really helpful - it allows us to expand our content and incorporate great new ideas.

Click here for an overview of the contributing process on GitHub. It will explain how open source contributing works and how you can get started.

Suggest a change

Have and idea about something that could be improved, but can't make the change yourself? No worries, you can still help.

If you have a GitHub account, you can go to the Issue Tracker page of the Introduction to Computer Science repository to see TO-DOs, feature requests, suggestions, and issues. To create a suggestion, click the 'New Issue' button.

If you don't have a GitHub account, you can send us an email at bitwisr@gmail.com. We will try our best to read and consider any ideas you have.
